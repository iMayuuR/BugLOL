# BugLOL: The Ultimate Error Sound Effect Extension 🚨

BugLOL is a VS Code extension that automatically plays a sound effect whenever your code hits an error. Turn your frustrating debugging sessions into instant comedy!

## Features

- **Massive Sound Library**: Search through 100,000+ meme sounds and sound effects!
- **Zero Bloat**: Downloads and caches only the sound you select. No heavy audio files bundled.
- **Trending Sounds**: Opens with the hottest trending regional memes by default.
- **Live Search**: Type to instantly search and discover new sounds.
- **Full Playback**: Plays the complete audio — no cut-offs!

## Usage

1. Install the extension.
2. The default error sound **"Fahhh"** is auto-downloaded on first run.
3. Open the Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`).
4. Type `Select Error Sound Effect` and hit Enter.
5. Pick any trending meme or search for a specific sound.
6. Write some buggy code and enjoy the show!

## Development

To test the extension locally:
1. Open this repository in VS Code.
2. Press `F5` to open the Extension Development Host.
